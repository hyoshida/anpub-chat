# anpub-chat

Anonymous that real name public chat.

## Requirements

* Node.js 0.11
* MongoDB 2.2

## Usage

1. Install libraries

  ```bash
  npm install
  bower install # (or `node_modules/.bin/bower install`)
  ```

2. Run a server

  ```bash
  npm start
  ```

3. Plese open `http://127.0.0.1:8000/` in your browser.
